#Основная часть#
В программе реализовал возможность пользователю через консоль вводить значения и делать операции (+, *, /, =) над значениями.

Так же во время выполнения кода, программа использует логирование действий и выводит лог на экран.

Программа создает список действий (лог) и записывает это в (CalculatorLog.txt) файл.
Путь: (/C#-ControlProject-OOP\Program\bin\Debug\net8.0/CalculatorLog.txt)
Есть вывод последних трех операций в консоль из этого файла.
Если файл превышает определенное количество строк чистит все что в нем находится (перезаписывает себя).

#Инструкция по установке# 
1. Для работы с программой вам нужно установить Visual Studio 2022.

2. После установки среды разработки вам нужно будет установить "Рабочие нагрузки" для студии:
2.1 Выберете (Create new project) в правой части панели Visual Studio 2022.
2.2 В правой части панели будет скрол, пролистайте в самый низ и выберете (Install more tools and fetures).
2.3 В открывшимся окне "Рабочие нагрузки" выберете следующие (Разработка классических приложений .NET) и
(Разработка приложений для универсальной платформы Windows) и установите их.
2.4 Установка завершена.

3. Теперь зайдя заново в Visual Studio 2022 вы можете выбрать (Open a project or solution) и выбрав
проект (файл с разрешением sln (в нашем случае C#-ControlProject-OOP.sln)) вы сможете запустить программу по характерной кнопке (Calculator) в верхней части приложения.
